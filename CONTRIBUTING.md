# Contributing to NixTio

Thank you for your interest in contributing to the NixTio Sector Intelligence Tracker!

## How to Contribute

1. **Fork the repository**
2. **Create a new branch** (`git checkout -b feat/amazing-feature`)
3. **Commit your changes** following [Conventional Commits](https://www.conventionalcommits.org/)
4. **Push to the branch** (`git push origin feat/amazing-feature`)
5. **Open a Pull Request**

## Development Setup

See the [README.md](README.md) for installation instructions.

## Code Style

- Follow PEP 8 for Python code.
- Use meaningful variable and function names.
- Document complex logic with comments.
